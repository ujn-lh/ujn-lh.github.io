id name class的区别
Created 2020-03-25| Updated 2020-03-25|Web
Post View:

1.Id

            Id是标签的唯一标识，类似于我们的身份证号，一个页 面中每个Id只能用一次，用于区分不同的结构和内容，常 用于后台。

2.Name

            Name是标签的名称，和我们的名字一样是可以重复的。

3.Class

           Class将标签分成众多类别，主要用来设置Css样式，拥有 相同样式的标签往往在同一类里面，而且一个标签可属于很 多个类。
