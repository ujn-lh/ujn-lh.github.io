Java Bean

Posted on 2020-03-25 | In JSP

Java Bean 在 JSP 中的作用。

           使用JavaBean的目的是为了给JSP“瘦身”，将JSP拆分为JSP+JavaBean，将之前只用JSP完成的功能，交由JSP和JavaBean共同完成。目的是让JSP只关注“显示”，让JavaBean和数据打交道。

Java Bean 的设计原则

  • 方法都是公有的
  • 属性是私有的
  • 无参的构造方法
  • 提供getter和setter方法

普通方式访问

使用useBean动作

Java Bean 的四个作用域范围。(可以设置在 useBean 的作用范围 scope中)

  • page:仅在当前页面有效。
  • request:可以通过request.getAttribute()取得Bean对象。
  • session:可以通过session.getAttribute()取得Bean对象。
  • application:可以通过application.getAttribute()取得Bean对象。
