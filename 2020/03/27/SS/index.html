CSS基本结构

①selector{②property:③value;}

①选择器：表明花括号中的属性设置将应用于哪些HTML元素，例如“body”

②属性：例如用于设置背景色的属性“background-color”等等

③值：比如“background-color”属性的值可以是“FF0000”代表红色



声明:
  • 由属性和值对组成，在一个规则中可以有不止一个声明
  • 每个声明都必须以分号结束，表示与下一个声明隔开
  • 如果省略了分号，该声明和下条声明都会被忽略
  • 规则中包含的花括号和所有声明常被统称为声明块
  • CSS忽略声明块中的空白和回车，通常将块中的每条声明写到单独的行中
  • 后面的规则优先于前面
  • 通常每行只描述一个属性，这样可以增强样式定义的可读性

CSS选择器

           CSS的思想就是首先指定对什么“对象”进行设置，然后指定对该对象的哪个方面的“属性”进行设置，最后给出该设置的“值”

  • 元素选择器

    •            又称类型选择器(或标记选择器)即：文档的元素就是最基本的选择器 ,作用就是声明哪些元素采用哪种CSS样式。


    CSS对所有的属性和值都有相对严格的要求，如果出错，语句无效.
  • 类选择器

    •            类（class）选择器允许以一种独立于文档元素的方式来指定样式， 要应用样式而不考虑具体设计的元素，最常用的方法就是使用类选择器， 为了将类选择器的样式与元素关联，必须将元素中的class属性指定为一个适当的值 ,在类选择器中，类名用句点 “.” 表示；

               类选择器可以单独使用，也可以结合元素选择器使用


               在HTML中，一个class属性还可能包含多个属性值，各个值之间用空格分隔，用于将多个类的样式同时应用到一个标记中


  • ID选择器

    •            ID选择器是通过引用 id 属性的值来选定目标

               ID选择器的标志是前面有一个 “#” 字符

    与类选择器的区别

               与类选择器不同，在一个 HTML 文档中，ID选择器仅使用一次。不同于类选择器，ID选择器不能结合使用，因为ID属性不允许有以空格分隔的多个值 。ID选择器能包含更多含义，类似于类，可以独立于元素来选择ID 。


  • 组合选择器

    •            组合选择器是多个选择器(包括元素选择器、class选择器、ID选择器等)通过逗号连接而成的 用于把风格完全相同的选择器组合在一起，以得到简洁的样式表 h1, h2, .special, #intro {color: blue; font-family: Arial}


  • 通用选择器

    •            该选择器可以与任何元素匹配，就像是一个通配符 ，通用选择器用一个星号来表示 ，例如，下面的规则可以使文档中的每个元素都为红色： * {color: red;} 上述声明等价于列出了文档中所有元素的一个组合选择器。利用通用选择器，只需敲一个星号就能使文档中所有元素的color属性值指定为red

字体属性

           CSS中，字体样式通过一个与字体相关的属性系列来指定。

  • 字体(font-family)

    •            在 CSS 中，有两种不同类型的字体系列名称：通用字体系列 - 拥有相似外观的字体系统组合（比如 "Serif" 或 "Monospace"）,特定字体系列 - 具体的字体系列（比如 "Times" 或 "Courier"）

  • 大小(font-size)

    •            属性名称：font-size.

               font-size值可以是绝对或相对值.

               绝对值：将文本设置为指定的大小。不允许用户在所有浏览器中改变文本大小（不利于可用性）。绝对大小在确定了输出的物理尺寸时很有用。

               相对大小：相对于周围的元素来设置大小。允许用户在浏览器改变文本大小。避免在网页样式表中使用绝对单位。

               相对单位em：与当前字体尺寸相等(即给定字体的font-size值)的计量单位。ex：大约等于所用字体中小写字母x的高度，浏览器实现时1ex=0.5em。px：像素，也称作相关像素。%：百分比值。

               当指定单位时，确保单位紧跟在数字后面，之间没有间隔。例：h1{font-size:1.5em;}。在大多数当前的浏览器中可以使用绝对关键字。例：h1{font-size: x-large;}

               使用相对关键字来调整文字尺寸，使之比周围文字更大或更小。例：h1{font-size: larger;}。作为百分比值，在元素默认或继承的基础上增加和减小尺寸。例：h1{font-size:150%;}。注意：如果没有规定字体大小，普通文本（比如段落）的默认大小是 16 像素 (16px=1em)。

  • 粗细(font-weight)

    •             属性名称：font-weight。

               性值：normal | bold | bolder | lighter | 100 | 200 | 300 | 400 | 500 | 600 | 700 | 800 | 900 | inherit。

               默认值：normal。

               适用对象：所有元素。

               是否可继承：是。

               网页中大多使用的普通字体只有两个字重：正常（normal）和粗体（bold）； 数字表示法很少使用；

               值400对应于normal，700对应于bold，600及以上通常表示为粗体。

  • 样式(font-style)

    •            属性名称：font-style

               属性值：normal | italic | oblique | inherit。影响字体的形态是垂直的，还是倾斜的。italic和oblique都是斜体字

               不同之处：italic样式设计为字体弯曲；oblique只是把普通字体倾斜。通常情况下，italic 和 oblique 文本在 web 浏览器中看上去完全一样。

               默认值：normal。

               适用对象：所有元素。

               是否可继承：是。

  • 变形(font-variant)

    •             属性名称：font-variant。

               属性值：normal | small-caps(小体大写字母 ) | inherit

               默认值：normal。

               适用对象：所有元素。

               是否可继承：是。

               如果font-variant属性被设置为small-caps，而没有可用的支持小体大写字母的字体，那么浏览器多半会将文字显示为正常尺寸（而不是小尺寸）的大写字母。

  • 快捷属性(font)

    •            CSS提供快捷字体属性把所有与字体相关的属性编译到一个规则中去。

               属性名称：font。

               属性值：font-style font-weight font-variant font-size/line-height font-family|inherit。

               默认值：由每个属性列表中的默认值决定.

               适用对象：所有元素。

               是否可继承：是。 font属性值是用空格隔开的，是所有字体相关属性值的列表。

               属性值的顺序很重要｛font: style weight variant size font-family｝。font属性的前三个值允许采用任意的顺序，而后两个值必须有且顺序是font-size在前，font-family在后；遗漏一个或者放错顺序将导致整个规则失效

文本属性

  • 属性名称：text-align

    •            属性值：left | right | center | justify | inherit.

               默认值：从左到右阅读的语言为left；从右到左阅读的语言为right；

               适用对象：块级元素和表格的单元格.

               是否可继承：是.

               在下例中，标题（th）中的文字被设置为右对齐，而表中数据（td） 被设置为居中。正常的文本段落被设置为两端对齐。 th { text-align: right; } td { text-align: center; } p { text-align: justify; }

  • 属性名称：text-indent

    •            使文本第一行缩进具体数目的长度

               属性值：长度，百分比值|inherit.

               默认值：0.

               适用对象：块级元素和表格的单元格，不能是内联元素.

               是否可继承：是.

               例： p#1{text-indent: 2em;} p#2{text-indent: 25%;} p#3{text-indent: -35px;}　　//悬挂缩进

  • 属性名称：line-height

    •            定义了文本中基线之间的最小距离

               属性值：数值，长度，百分比|normal|inherit.

               默认值：normal.

               适用对象：所有元素.

               是否可继承：是.

               例：用三种方法，将行高设置为字体尺寸的两倍 p{line-height: 2;} //数字作为当前字体尺寸要乘的比例因子 p{line-height: 2em;}　//以下两行都是基于当前字体的尺寸 p{line-height: 200%;}

  • 属性名称：word-spacing

    •            在显示的单词之间插入空格.

               属性值：长度值，normal.

               默认值：normal，与设置值为 0 一样.

               适用对象：所有元素.

               是否可继承：是.

               例： p.spread {word-spacing: 0.5em;} p.tight {word-spacing: -0.5em;} p.base {word-spacing: normal;} p.norm {word-spacing: 0;}

  • 属性名称：letter-spacing

    •            在显示的单词之间插入空格.

               属性值：长度值，normal | inherit.

               默认值：normal，与设置值为０一样.

               适用对象：所有元素.

               是否可继承：是.

               例： h1 { letter-spacing: 6px; } P { letter-spacing: 3px; }.

  • 属性名称：text-decoration

    •            属性值：none | underline | overline | line-through | blink | inherit.

               默认值：none.

               适用对象：所有元素.

               是否可继承：否，但是因为线也会通过子元素，所以它们看起来也被“修饰”了.

               该属性最常见的用法是取消链接自动出现的下划线：例：a{text-decoration: none;}.

               因为下划线是“点这里“的强烈视觉提示，所以给非链接文本添加下划线会产生误导和混淆；

  • 属性名称：text-transform

    •            CSS属性text-transform用于控制文本的大小写。无论字母本来的大小写，你可以通过该属性令它首字母大写（capitalize）、全部大写（uppercase）或者全部小写（lowercase）。

               属性值：none | capitalize | lowercase | uppercase | inherit.

               默认值：none.

               适用对象：所有元素.

               是否可继承：是.

               例： h1 {text-transform: uppercase;}
