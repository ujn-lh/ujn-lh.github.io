Java Database Connectivity
Created 2020-04-08| Updated 2020-04-09|JSP
Post View:

JDBC的基本介绍:

  • JDBC是Java程序与数据库系统通信的标准API
  • JDBC的实现由厂商以驱动程序的形式提供
JDBC中常用的接口和类:
  • Connection:与特定的数据库连接
  • DriverManager:管理一组JDBC驱动程序的基本服务
  • Statement:用于执行静态SQL语句并返回其生成结果的对象
  • PreparedStatement:集成Statement接口，表示预编译SQL语句的对象
  • ResultSet:值查询返回数据库结果集
  • ResultSetMetaData:可获得关于ResultSet对象中列的类型和属性信息对象。

JDBC的操作步骤:


  1. 注册JDBC驱动程序：
    Class.forname(“”);

  2. 建立到DB的连接：
    Connection conn=DriverManager.getConnection(url,”username”,”password”);

  3. 创建SQL语句：定义SQL语句,然后使用预编译的SQL语句。
    PreparedStatement ps=conn.prepareStatement(sql);

  4. 执行SQL语句：
    ①.ps.executeUpdate();
    ②.ps.executeQuery();
    有结果集
    ③.ResultSet rs=ps.executeUpdate();
    ④.ResultSet rs=ps.executeQuery();

  5. 处理结果(有的话)：可以用next遍历等来进行操作。

  6. 与数据库断开连接：
    ①.rs.close();//结果集存在
    ②.ps.close();
    ③.conn.close();
